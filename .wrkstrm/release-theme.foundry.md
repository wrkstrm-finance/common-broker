# Release Theme (Foundry)

This file defines release codenames for CommonBroker. The theme leans on
brokerage infrastructure, routing, and settlement.

## Naming format

- Canonical form: `<version> - <Codename>`
- Codename format: Title Case, two words preferred.
- Vocabulary: routing, clearing, settlement, custody, risk.

## Major releases

- 1.0.0 - Clearing Rail
- 2.0.0 - Unified Ledger
